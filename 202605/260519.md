### [WORK] 오탈자 등록 기능
사용자가 도서 상세 페이지에서 오탈자(오류)를 직접 제보할 수 있는 등록 API 개발. 등록할 때 판/쇄, 위치, 유형 등 정보를 같이 받고, 존재하지 않는 페이지를 제보하지 못하도록 검증 로직 추가함 (CORE-745).

프론트엔드에서 본인인증 방식을 포트원 브라우저 SDK 기반으로 전환하는 작업 완료. 본인인증 성공 여부만 필요한 화면에서 재사용할 수 있게 공통 컴포넌트로 만들었고, 응답 상태별(성공/입력 오류/인증 불일치/미존재/중복 인증/PG 오류 등) 처리 흐름을 정리해서 테스트까지 확인함 (CORE-672).
