### [WORK] 결제 완료 API 설계
결제 완료 API는 주문번호와 결제 성공/실패 여부를 요청값으로 받아, 주문 내역 기준으로 마이콘텐츠 입과 처리, 주문 상태 변경, 마일리지 적립을 처리하도록 설계. 사용자가 결제 후 취소/환불하는 경우 종이책만 취소가 가능하며, 종이책은 배송 준비 전 단계에서만 구매 취소가 가능하다는 정책을 확인.
